## 总体流程概览

```
优化迭代循环:
  ↓
residual_jacobian_batch_quat() 被调用
  ↓
Step 1-4: 计算重投影损失的 grad_reproj 和 Hess_reproj
  ↓
Step 5: 计算深度损失的 grad_depth 和 Hess_depth  ← 本文重点
  ↓
Step 6: 融合 → grad_total = grad_reproj + grad_depth
           Hess_total = Hess_reproj + Hess_depth
  ↓
返回 (-grad_total, Hess_total) 给优化器
  ↓
优化器求解: (Hess_total + λI) · Δ = -grad_total
  ↓
更新位姿: T_new = T_old · exp(Δ)
```

## 深度损失的梯度计算

### 1. 深度残差定义

```python
# 代码位置: costs.py 第465-466行
pred_depth = depths[0]  # 当前位姿预测的深度（米）
residual = pred_depth - gt_depth  # 深度残差（米）
```

**数学表示**:
\[
r_{depth} = d_{pred} - d_{gt}
\]

其中：
- \(d_{pred}\)：从DSM预测的深度（通过ray casting计算）
- \(d_{gt}\)：真实深度（ground truth）

### 2. 数值微分计算Jacobian矩阵

```python
# 代码位置: costs.py 第484-490行
J_depth = np.zeros(6)  # 6维：3个旋转参数 + 3个平移参数
eps = 1e-4  # 微小扰动

for j in range(3):  # 只对旋转的3个参数计算梯度
    # 对第j个旋转参数添加微小扰动
    pred_depth_perturbed = depths[j + 1]  # 扰动后的深度
    J_depth[j] = (pred_depth_perturbed - pred_depth) / eps
```

**数学表示**:
\[
J_{depth}[j] = \frac{\partial d_{pred}}{\partial \theta_j} \approx \frac{d_{pred}(\theta + \epsilon \cdot e_j) - d_{pred}(\theta)}{\epsilon}
\]

其中：
- \(\theta\)：当前位姿参数（旋转+平移，共6维）
- \(\epsilon\)：微小扰动（旋转参数用 1e-4 度，高度用 0.5 米）
- \(e_j\)：单位向量（第j维为1，其他为0）
- 注意：当前实现计算**高度（alt）和旋转参数（pitch, roll, yaw）的梯度**
  - 高度梯度：放在 `J_depth[5]`（对应平移的Z分量）
  - 旋转梯度：放在 `J_depth[0,1,2]`（对应 pitch, roll, yaw）

### 3. 应用鲁棒损失函数

```python
# 代码位置: costs.py 第477-482行
depth_cost_tensor = torch.tensor([residual ** 2], ...)
depth_loss, depth_w_loss, _ = self.loss_fn1(
    depth_cost_tensor,
    alpha=0.0,  # 使用对数损失
    truncate=depth_truncate  # 截断值，例如2.0米
)
```

**鲁棒损失函数**（alpha=0时）:
\[
\rho(r^2) = 2 \cdot \text{truncate}^2 \cdot \log\left(1 + \frac{r^2}{2 \cdot \text{truncate}^2}\right)
\]

**权重导数**:
\[
w_{depth} = \rho'(r^2) = \frac{2}{r^2 / \text{truncate}^2 + 2}
\]

作用：大误差的权重会自动降低，但仍提供梯度方向。

### 4. 计算深度梯度

```python
# 代码位置: costs.py 第496行
grad_depth[0, original_i] = J_depth_torch * residual * depth_w_loss.item() * depth_weight
```

**数学表示**:
\[
g_{depth} = J_{depth}^T \cdot r_{depth} \cdot w_{depth} \cdot \lambda_{depth}
\]

其中：
- \(J_{depth}^T\)：深度Jacobian的转置（6×1）
- \(r_{depth}\)：深度残差（标量）
- \(w_{depth}\)：鲁棒损失权重（标量）
- \(\lambda_{depth}\)：深度权重超参数（例如0.2）

**维度**：`grad_depth` 形状为 `[1, num_init_pose, 6]`

## 深度损失的Hessian计算

### Gauss-Newton近似

```python
# 代码位置: costs.py 第497行
Hess_depth[0, original_i] = torch.outer(J_depth_torch, J_depth_torch) * depth_w_loss.item() * depth_weight
```

**数学表示**:
\[
H_{depth} = J_{depth}^T \cdot J_{depth} \cdot w_{depth} \cdot \lambda_{depth}
\]

其中：
- \(J_{depth}^T \cdot J_{depth}\)：Gauss-Newton近似（6×6矩阵）
- \(w_{depth}\)：鲁棒损失权重
- \(\lambda_{depth}\)：深度权重超参数

**维度**：`Hess_depth` 形状为 `[1, num_init_pose, 6, 6]`

**Gauss-Newton近似的原理**：
- 忽略残差的二阶导数
- 假设损失函数接近二次型
- 计算效率高，通常效果良好

## 融合到重投影损失

### 梯度融合

```python
# 代码位置: costs.py 第524行
grad = grad + grad_depth
```

**数学表示**:
\[
g_{total} = g_{reproj} + g_{depth}
\]

其中：
- \(g_{reproj}\)：重投影损失的梯度（来自特征匹配）
- \(g_{depth}\)：深度损失的梯度

**维度**：`[1, num_init_pose, 6]`

### Hessian融合

```python
# 代码位置: costs.py 第525行
Hess = Hess + Hess_depth
```

**数学表示**:
\[
H_{total} = H_{reproj} + H_{depth}
\]

其中：
- \(H_{reproj}\)：重投影损失的Hessian
- \(H_{depth}\)：深度损失的Hessian

**维度**：`[1, num_init_pose, 6, 6]`

### 返回给优化器

```python
# 代码位置: costs.py 第527行
return -grad, Hess, w_loss, valid_query, p2d_q, cost
```

**注意**：返回的是 **-grad**（负梯度），因为优化器求解的是：
\[
H \cdot \Delta = -g
\]

## 优化器如何使用

### Levenberg-Marquardt算法

```python
# 伪代码（来自 optimizer_step 函数）
def optimizer_step(g, H, lambda_=0.1):
    # 1. 添加阻尼项（LM算法）
    H_damped = H + lambda_ * diag(H) * I
    
    # 2. 求解线性系统
    # (H + λD) · Δ = -g
    delta = solve(H_damped, -g)
    
    # 3. 更新位姿
    T_new = T_old · exp(Δ)
    return delta
```

### 完整的优化步骤

1. **构建线性系统**:
   \[
   (H_{total} + \lambda D) \cdot \Delta = -g_{total}
   \]
   
   其中：
   - \(H_{total} = H_{reproj} + H_{depth}\)
   - \(g_{total} = g_{reproj} + g_{depth}\)
   - \(\lambda\)：LM阻尼系数（通常0.1）
   - \(D\)：Hessian的对角线

2. **求解增量**:
   - 使用Cholesky分解（如果正定）
   - 否则使用LU分解

3. **更新位姿**:
   \[
   T_{new} = T_{old} \cdot \exp(\Delta)
   \]
   
   其中 \(\Delta = [\Delta\omega, \Delta t]\)（旋转+平移）

## 数学公式总结

### 完整的目标函数

\[
\min_{\theta} \quad L_{total} = L_{reproj}(\theta) + \lambda_{depth} \cdot L_{depth}(\theta)
\]

其中：
- \(L_{reproj}(\theta) = \sum_i w_i \cdot \rho(r_i^2(\theta))\)
- \(L_{depth}(\theta) = w_{depth} \cdot \rho(r_{depth}^2(\theta))\)

### 梯度

\[
g_{total} = \frac{\partial L_{total}}{\partial \theta} = g_{reproj} + \lambda_{depth} \cdot J_{depth}^T \cdot r_{depth} \cdot w_{depth}
\]

### Hessian（Gauss-Newton近似）

\[
H_{total} = \frac{\partial^2 L_{total}}{\partial \theta^2} \approx H_{reproj} + \lambda_{depth} \cdot J_{depth}^T \cdot J_{depth} \cdot w_{depth}
\]

### 优化步骤

\[
\Delta = -(H_{total} + \lambda D)^{-1} \cdot g_{total}
\]

## 💡 关键要点

1. **深度约束通过梯度融合**：深度损失不是单独优化，而是与重投影损失**相加**，形成统一的目标函数。

2. **数值微分**：由于深度预测函数很复杂（涉及ray casting），使用**数值微分**计算Jacobian，而不是解析求导。

3. **只优化旋转参数**：当前实现只对旋转参数（pitch, roll, yaw）计算梯度，平移参数梯度为0。

4. **鲁棒损失函数**：使用对数损失函数自动降低大误差的权重，但仍提供梯度方向。

5. **权重平衡**：`depth_weight=0.2` 控制深度约束和重投影约束的相对重要性。

6. **Gauss-Newton近似**：Hessian使用一阶近似，计算高效且通常效果良好。

## 🔍 调试建议

观察以下指标判断深度约束是否生效：

1. **梯度范数比例**：
   ```
   grad_depth_norm / grad_reproj_norm
   ```
   理想范围：0.1 - 1.0

2. **Hessian范数比例**：
   ```
   Hess_depth_norm / Hess_reproj_norm
   ```
   理想范围：0.1 - 1.0

3. **深度误差统计**：
   - 误差中位数是否减小
   - 误差≤1m的位姿比例是否增加
